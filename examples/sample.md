# Markdown to HTML Converter Sample Document

Welcome to the live preview of the Markdown to HTML Converter desktop application! This sample document showcases all the supported Markdown syntax elements that are manually parsed and generated into clean HTML.

## Text Styling & Formatting

You can style text easily using standard Markdown markers:
- **Bold text** using `**Bold text**`
- *Italic text* using `*Italic text*`
- ***Bold and Italic text*** using `***Bold and Italic text***`
- `Inline code block` using backticks: `` `Inline code block` ``
